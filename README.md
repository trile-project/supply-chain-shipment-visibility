#Supply Chain Shipment Visibility Dashboard

**Project Overview**

#This project analyzes shipment execution performance across carriers, modes, and regions to measure service reliability and identify root causes of late deliveries.

#The objective is to simulate a real supply chain analyst workflow used in large logistics and e-commerce companies.

#Business Problem: Late shipments negatively impact customer experience and logistics cost. This project builds visibility into service performance and identifies operational improvement opportunities.

#Tools Used
-SQL (MySQL)
-Excel
-Power BI

#Key KPIs
-On-Time Delivery %
-Late %
-Average Lead Time
-Carrier Performance
-Exception Analysis
-Cost vs Service tradeoff

#Analysis Performed
-Carrier scorecard
-Mode performance comparison
-Regional service analysis
-Root cause identification for late shipments
-Cost vs service evaluation

#Dashboard

-See /images folder for visuals.

#Key Insights

Example insights:
-Certain carriers show high volume but lower service reliability
-Ocean shipments drive the highest lateness risk
-Warehouse backlog is a top exception driver
-Priority shipments show different cost vs service tradeoffs

**Business Recommendations**
-Shift priority volume toward high-OTD carriers
-Improve exception monitoring using control tower alerts
-Standardize service level targets by mode
-Implement proactive late risk detection
